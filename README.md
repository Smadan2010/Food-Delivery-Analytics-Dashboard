**Food Delivery Analytics Dashboard**

An interactive business analytics dashboard developed to monitor food delivery operations, customer behavior, delivery performance, cancellations, and revenue insights using real-time visualizations and KPI tracking.

**Project Overview**

This project analyzes food delivery data to generate business insights and performance monitoring through an interactive Streamlit dashboard.

**The dashboard helps in:**

Monitoring revenue and order trends
Analyzing delivery performance
Identifying cancellation patterns
Tracking customer and restaurant ratings
Understanding top-performing cuisines and restaurants
Visualizing geographic delivery insights
